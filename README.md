# 📦 Inventory App

Aplikasi web sederhana untuk mengelola stok barang, mencatat barang masuk, pengambilan barang, dan melihat riwayat transaksi.

Project ini dibangun menggunakan **Next.js**, **TypeScript**, **Prisma ORM**, **PostgreSQL**, dan **NextAuth.js**.

---

## ✨ Fitur

- 🔐 Login dan autentikasi admin
- 📊 Dashboard ringkasan inventaris
- 📦 Manajemen daftar barang
- ➕ Pencatatan barang masuk
- ➖ Pencatatan pengambilan barang
- 🔄 Pembaruan stok otomatis
- ⚠️ Informasi stok menipis
- 🕒 Riwayat transaksi berdasarkan tanggal
- 🔍 Pencarian barang
- 📱 Tampilan responsif untuk desktop dan mobile
